Getting Started

Creating new posts is simple:

  1. Add a new page to this database
  2. Fill in the Name and Slug properties
  3. Set status to "Published" when ready
  4. Your blog will automatically sync every 6 hours!

Markdown Features

Your blog supports rich formatting including:

  • Bold and italic text
  • Code snippets and code blocks
  • Images and media
  • Dark mode toggle (click the theme button!)

    Your blog loads faster than Notion while keeping all the rich formatting you love!
